TGM-62095 4I4D2R 1AP_420 1OP_AL Timer
แสดงผลด้วยตัวเลข 7 SEGMENT สีแดง 
Sec : แสดงจำนวนเวลา
Bar : แสดงค่าที่รับได้จาก Input 4-20 mA
สามารถตั้งค่า Set Point เพื่อเริ่มต้นการตรวจจับค่า Max
Alarm : ทำงานเมื่อมีค่า หน้าจอมากกว่าหรือเท่ากับ Set Point 
Set Point Timer : สำหรับกำหนดระยะเวลาก่อนที่จะเกิดการทำงานของ Alarm
Set Point Pressure : สำหรับกำหนดจุดที่ต้องการเริ่มนับเวลา
สามารถกด Reset ผ่าน Input Contact ได้
ตั้งค่าการทำงานด้วย Remote IF
ตัวอย่าง  เช่น  ตั้ง set point pressure 15bar เมื่อ pressure
ถึง set point pressure ให้เวลาเริ่มนับจนกว่า pressure จะถึงเวลาที่ตั้งไว้
แล้วสั่งให้ อาลามทำงานตามเวลาที่ set point timer
ถ้าหากในระหว่างที่เวลายังไม่ถึงค่า set point pressure
แล้ว pressure ต่ำกว่า set point pressure ให้เวลาเริ่มนับใหม่จาก 0 
หาก pressure ขึ้นไปโดยไม่ตกให้เวลานับไปจนกว่าจะครบค่าที่ set point timer
แล้วสั่งให้ อาลามทำงานพร้อมกับ hold ค่าเวลาและ pressure ค้างไว้ที่ค่าสูงสุด
จนกว่าจะมากดปุ่ม Reset ให้เป็น0เพื่อเริ่มทำงานใหม่
