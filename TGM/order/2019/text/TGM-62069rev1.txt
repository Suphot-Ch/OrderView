รับอินพุต RS485 Modbus RTU

RS485  ตั้งค่า Baud Rate ได้ 4800 9600 19200 38400 57600
RS485  ตั้งค่า Parity ได้ None Odd Even
RTU     รองรับ Function Read Register 03h 04h
RTU     รองรับ Function Write Register 10h
