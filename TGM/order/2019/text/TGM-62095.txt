TGM-62095 4I4D2R 1AP_420 1OP_AL Timer
แสดงผลด้วยตัวเลข 7 SEGMENT สีแดง 
Sec : แสดงจำนวนเวลา
Bar : แสดงค่าที่รับได้จาก Input 4-20 mA
สามารถตั้งค่า Set Point เพื่อเริ่มต้นการตรวจจับค่า Max
Alarm : ทำงานเมื่อมีค่า หน้าจอมากกว่าหรือเท่ากับ Set Point
สามารถกด Reset ผ่าน Input Contact ได้
ตั้งค่าการทำงานด้วย Remote IF
ตัวอย่าง  เช่น  ตั้ง setting 15bar เมื่อ pressure
ถึง15 bar ให้เวลาเริ่มนับจนกว่า pressure เริ่มตกให้holdค่าสูงสุดค้างไว้แล้วเวลาก็หยุดที่ค่าสูงสุด
จนกว่าจะมากดปุ่มResetให้เป็น0เพื่อเริ่มทำงานใหม่
