TGM-62049 2I6D1R Timer Clock
แสดงผลด้วยตัวเลข 7 SEGMENT สีแดง 2.3"
แสดงเวลา [HH : MM : SS] [ชั่วโมง : นาที : วินาที] [ 00:00:00 - 99:59:59 ]
เป็นนาฬิกาจับเวลา
	เมื่อ แตะ Contact Input ค้าง นาฬิกาจะเริ่มนับขึ้นทุกๆ 1 วินาที และเมื่อปลด Contact Input ออก นาฬิกาจะหยุดนับเวลา
	เมื่อ แตะ Contact Reset 1 ครั้ง นาฬิกาจะถูก Reset เป็น 00:00:00
สามารถตั้งค่า Set Point Alarm ได้ 1 Output
สามารถตั้งค่า/แก้ไขเวลา ได้ด้วยรีโมท
ตั้งค่าการทำงานด้วย Remote IF
