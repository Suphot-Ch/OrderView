- Size 4" สีแดง
- Input Contact Switch 2 channel ( Start Stop )
- สามารถตั้งค่าการทำงานผ่าน Remote ได้
- Input Supply 220VAC 
- Output Supply 12VDC/2A
- แสดงค่า Counter Timer
- เมื่อ Trig Contact Start 1 ครั้ง Counter จะนับขึ้นทุกๆ 1 วินาที

และแสดงค่าเดิมค้างจนกว่าจะ Trig Contact Start อีกครั้ง 
จึงจะทำการ Reset Counter เป็น 0 และเริ่มนับใหม่อีกครั้ง
