แสดงเวลา [MM : SS] [นาที : วินาที]   [00 : 00] - [99 : 59]
เป็นนาฬิกาจับเวลา ตั้งนับขึ้นหรือนับลงได้
	เมื่อ แตะ Contact Input1 ค้าง นาฬิกาจะเริ่มนับขึ้นทุกๆ 1 วินาที และเมื่อปลด Contact Input 1 ออก นาฬิกาจะหยุดนับเวลา
	เมื่อ แตะ Contact Input2 ค้าง นาฬิกาจะเริ่มนับลงทุกๆ 1 วินาที และเมื่อปลด Contact Input 2 ออก นาฬิกาจะหยุดนับเวลา
	หาก Input 1 และ Input 2 แตะพร้อมกันจะไม่นับ
	เมื่อ แตะ Contact Reset 1 ครั้ง นาฬิกาจะถูก Reset เป็น ค่า Parameter เริ่มต้น สามารถตั้งได้
สามารถตั้งค่า Set Point ได้ 3 ช่อง
สามารถตั้งค่า/แก้ไขเวลา ได้ด้วยรีโมท
